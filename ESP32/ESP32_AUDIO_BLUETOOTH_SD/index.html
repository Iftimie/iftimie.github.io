ESP32 Bluetooth + SD Audio Pipeline

A set of related ESP32 experiments that take a WAV file, convert it to raw PCM, load it on the ESP32-CAM SD card, and stream it to Bluetooth headphones.

ESP32-CAM A2DP SD_MMC Audio

Open group folder

Pipeline summary

  • Prepare audio: `out.wav` is converted to raw PCM (`out2.raw`) at 44.1 kHz stereo s16le.
  • Upload: copy `out2.raw` to the SD card root on the ESP32-CAM.
  • Stream: A2DP source reads raw bytes from SD and streams to Bluetooth headphones.
  • Variants add connection tests, SD listing, and play/pause control.

Implementation notes

  • All variants use `BluetoothA2DPSource` with a `raw_cb` callback feeding bytes.
  • SD_MMC is mounted in 1-bit mode on ESP32-CAM for reliability.
  • The pause variant gates playback with `playEnabled`, zero-fills when stopped, and restarts on demand.
  • Python helpers (`main.py`, `gibberish.py`) preprocess audio into the raw format.

Example conversion: ffmpeg -i out.wav -ac 2 -ar 44100 -f s16le out2.raw
