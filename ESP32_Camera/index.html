ESP32-CAM Streaming Suite

A family of ESP32-CAM sketches for bring-up, local web streaming, and pushing frames to a server via HTTP or WebSockets.

ESP32-CAM Streaming WiFi JPEG / RAW

Open project folder

How the variants relate

  • Simple Setup verifies pinout, PSRAM, and capture reliability.
  • Camera WebServer provides a local web UI and live stream.
  • HTTP Upload captures grayscale frames, converts to JPEG, and POSTs to a server.
  • WebSocket JPEG pushes JPEG frames over a WS binary channel.
  • WebSocket Raw sends raw grayscale frames with a width/height header.

Implementation notes

  • All streaming variants use esp_camera_fb_get() and return buffers promptly.
  • JPEG conversion uses fmt2jpg() on grayscale frames (quality ~30).
  • WebSocket RAW includes an 8-byte header (width + height) before pixel data.
  • WiFi reconnect logic is basic; restart is used on timeouts.
