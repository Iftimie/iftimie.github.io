# PIC10F200 Button Toggle

Button-driven LED toggle inspired by the CircuitBread PIC10F200 button inputs tutorial.

## What it does
- Reads a button input and toggles an LED output on the PIC10F200.
- Based on the CircuitBread tutorial:
  https://www.circuitbread.com/tutorials/button-inputs-part-9-microcontroller-basics-pic10f200

## Hardware
- PIC10F200
- Button + LED

## Software / Tools
- MPLAB / assembler

## How to run
- Build in MPLAB/MPASM and program the PIC10F200.
