# PIC10F200 Charlieplex LEDs

Charlieplexed LED patterns inspired by the CircuitBread PIC10F200 Christmas lights tutorial.

## What it does
- Drives a charlieplexed LED array with timed patterns on a minimal pin budget.
- Based on the CircuitBread tutorial:
  https://www.circuitbread.com/tutorials/christmas-lights-special-microcontroller-basics-pic10f200

## Hardware
- PIC10F200
- LED array

## Software / Tools
- MPLAB / assembler

## How to run
- Build in MPLAB/MPASM and program the PIC10F200.
