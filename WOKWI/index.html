Wokwi LED Cube - 16x16x16 Journey

I wanted to build a 16x16x16 "holographic" LED cube. The cube is really a stack of 2D LED matrices, so I started small, simulated the driver stack, and then pushed toward real hardware.

Start small: 2x2 matrix

The first prototype was a tiny 2x2 LED matrix to validate the concept of stacking 2D layers.

Custom MBI5124 simulation

To simulate a 16-channel current sink (MBI5124), I built a custom Wokwi chip and a diagram generator that lays out two 16x16 layers (2x 16x16) for quick iteration.

  • src/mbi5124.chip.c models SDI/CLK/LE/OE and 16 sinking outputs.
  • diagram_generator.py builds the Wokwi JSON with two layers and daisy-chained MBIs.
  • .dev-container/Dockerfile sets up the clang+wasm builder and Arduino CLI.

Build steps (from README): docker build -t wokwi-mbi5124-led-matrix-dev -f .dev-container/Dockerfile .

Two-layer Wokwi simulation (2 x 16x16).

PCB design: layer + motherboard

EasyEDA was not ideal for repeated or programmatic work, so the layer and motherboard layouts were done manually. The layer boards were designed first, then the motherboard.

Hardware tests

The first hardware test worked on the first try using the code from REAL_LED_MATRIX_NANO. One layer looked great, but at five layers the LED density and thick PCB edges made the cube hard to read.
